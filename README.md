# JShupu
#### 脚本名称
安静看虎扑 又名 看球吃瓜小神器
#### 脚本由来
1. 个人并不喜欢APP也不喜欢注册，但喜欢看球吃瓜。
2. 打工人只能看看球这样来摸鱼。
3. 不想 **注册不想下载APP**，安静看手机虎扑。
4. 随着某扑的更新，页面越来越多三无广告、各种限制，只希望你使用APP打开。
#### 脚本功能
1. 去除**首页广告**；
2. 去除**首页APP打开**;
3. 去除**首页浏览APP的MASK;**
4. 去除**帖子内广告**；
5. 去除**帖子内一堆无意义APP诱导阅读**；
6. 去除**帖子内查看回复需登录需下载APP限制**；
7. 去除**帖子内视频试看59秒限制**；
